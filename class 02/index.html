Which HTML Tag is an inline element and which HTML Tag is a block element?

In HTML, elements are categorized as either inline or block elements based on their default display behavior.

### Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples include:

  • "- <'span'>"
  • "- <'a'>"
  • "- <'em'>"
  • "- <'br'>"
  • "- <'strong'>"
  • "- <'img'>"
  • "- <'input'>"
  • "- <'button'>"
  • "- <'label'>"
  • "- <'select'>"
  • "- <'textarea'>"
  • "- <'code'>"
  • "- <'cite'>"
  • "- <'abbr'>"

    • ### Block Elements

    Block elements start on a new line and take up the full width available. Examples include:

  • "- <'div'>"
  • "- <'p'>"
  • "- <'hr'>"
  • "- <'h1'> to <'h6'>"
  • "- <'ul'> and <'ol'>"
  • "- <'table'>"

    • It's important to understand the difference between inline and block elements as it can affect the layout and structure of your HTML document.
